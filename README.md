# 🌐 Personal Portfolio Website

A responsive personal portfolio website built using **ReactJS** to showcase my skills, projects, and contact information.

---

## 📌 Features

- 🏠 Home section with name, intro, and profile image  
- 👩‍💻 About section with description and skills  
- 📁 Projects section with project details  
- 📬 Contact section with form (Name, Email, Message)  
- 🧭 Navigation bar for smooth scrolling between sections  
- 📱 Fully responsive design for all devices  

---

## 🛠️ Tech Stack

- ReactJS  
- JavaScript (ES6+)  
- HTML5  
- CSS3  
- (Optional: Bootstrap / Tailwind CSS)
